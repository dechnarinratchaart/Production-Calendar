ต้องการ Google Calendar API access
ดูวิธีตั้งค่า API

วิธีตั้งค่า Google API

เพื่อให้ระบบนี้ทำงานได้ต้องสร้าง Google OAuth Client ID ก่อนครับ

  1. ไปที่ Google Cloud Console
  2. สร้าง Project ใหม่ หรือเลือก Project ที่มีอยู่
  3. เปิดใช้งาน Google Calendar API
  4. ไปที่ Credentials → Create Credentials → OAuth Client ID
  5. เลือก Application type: Web application
  6. เพิ่ม Authorized JavaScript origins: URL ที่คุณ host เว็บนี้
  7. คัดลอก Client ID มาใส่ในไฟล์ HTML บรรทัด CLIENT_ID
